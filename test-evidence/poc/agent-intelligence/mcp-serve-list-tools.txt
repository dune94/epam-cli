$ node dist/epam.js mcp serve --list-tools

EPAM CLI MCP Tools:

  read_file
    Read the contents of a file at the given path.
  write_file
    Write content to a file. Creates parent directories if needed.
  bash
    Execute a bash command. Use for running scripts, installing packages, running tests, etc.
  list_files
    List files matching a glob pattern. Returns a list of matching file paths.
  search
    Search for a pattern in files using ripgrep (rg) or grep. Returns matching lines with file and line number.
  fetch_url
    Fetch the content of a URL and return it as text.
